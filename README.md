# IntelliNote

IntelliNote is an AI-powered note generation platform built using the MERN stack. The application helps students generate exam notes, diagrams, charts, and revision material using AI.

The project includes authentication, a credit-based usage system, and payment integration for premium features.

## Features

* Generate AI-based exam notes
* Create diagrams, graphs, and charts automatically
* Revision mode for short and quick notes
* Class-wise and subject-wise important questions
* Credit-based usage system
* Stripe payment integration
* Secure authentication using JWT
* Google Authentication using Firebase

---

## Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
AI: Gemini API
Payments: Stripe
Authentication: JWT & Firebase Auth
Deployment: Render

---

## How It Works

1. User logs in using Google authentication.
2. The system assigns free credits.
3. Users enter a topic or subject.
4. AI generates structured notes, diagrams, and revision material.
5. Credits decrease based on usage.
6. Users can purchase additional credits using Stripe.

---

## Purpose

This project was built as a full-stack MERN application to explore building a real AI-powered SaaS product with authentication, payments, and production-level backend APIs.
